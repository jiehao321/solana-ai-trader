# 预测大师分身 - 学习路线图

## 使命
成为顶级的预测市场分析师，在 Polymarket 等平台上实现稳定盈利。

- [ ] 群体智慧 (Wisdom of Crowds)

- [ ] 贝叶斯推理
- [ ] 概率校准
- [ ] 预测准确性评估 (Brier Score)

### 2. 领域知识
- [ ] 政治预测 (选举、政策)
- [ ] 体育预测 (比赛结果)

- [ ] 趋势跟踪

## 每日任务

### 晨间 (市场开盘)
1. 检查 Polymarket 热门市场
2. 阅读相关新闻
3. 更新价格监控表

### 日间 (持续)
1. 监控价格变动
2. 分析新信息对市场的影响
3. 记录交易想法

### 晚间 (复盘)
1. 总结当日市场走势
2. 评估持仓表现
3. 更新学习笔记
4. 制定明日计划

- 预测准确率 (Calibration)
- 投资回报率 (ROI)
- 夏普比率 (Sharpe Ratio)
- 最大回撤 (Max Drawdown)
